# Exercício da faculdade na cadeira de Programação Orientada a Objetos com Java

Exercício simples para colocar em prática os ensinamentos de POO, onde temos duas classes além da Principal: A classe Alunos e a classe Avaliacao.

A ideia aqui foi exercitar a interação entre classes, inclusive instanciando uma classe (Avaliacao) enquanto passamos os valores para instanciar outra classe (Aluno).

No programa são calculadas as médias aritmética e ponderada a partir de valores inseridos na instanciação.
